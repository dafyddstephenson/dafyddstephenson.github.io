About

I currently work in the Climate and Global Dynamics division of the National Center for Atmospheric Research, Colorado, USA. My research is part of a NASA project to establish links between atmospheric and oceanic fluctuations in the climate system, which I achieve through state-of-the-art global (adjoint) modelling and the development of novel data analysis methods.

Climate-driving weather patterns

By developing a new optimisation method, my current work combines decades of Earth observations with simulated physics to find the atmospheric patterns causing the most noise in climate change trends. See this recent conference talk.

Antarctic Intrusions

After we noticed messy spikes in shipboard data, I was awarded a secondment project identifying their origins using thousands of measurements from satellite images to free-drifting "robot" floats. Here's a poster with the details.

Album cover colormaps

Just for fun, using machine learning to create colormaps for data visualisation. My script searches Wikipedia for an album's artwork then makes the colour scale using K-Means. Add them to your own matplotlib tool belt or make me a suggestion on Twitter.

Adjoint modelling resources

Anything I have set up for my research is as public as I can make it. Adjoint models are powerful but underutilised tools for answering questions about the sensitivity of the climate system. In this seminar, I open with a 15 minute intro to what these things are and how I use them. I also wrote this Jupyter Notebook which takes a simple "Pringles tube" model of an ocean gyre and calculates its adjoint for the user to play/learn with. Feel free to use it for teaching.

If you have access to a supercomputer (cloud versions are TBA) and would like to run either of the adjointed global circulation models I have used in my research, I'd be happy to help you set them up. Here is everything I needed to set up NEMO's (freeware) adjoint model NEMOTAM. It's a lot, so just send me an email if you're stuck. MITgcm's adjoint model is (currently) not freely available, but if you have the licence money, here are my notes for setting it up on UCAR's Cheyenne and NASA's Pleiades HPC systems. There's also a lot documented in there you can do to hack the ECCO state estimate without needing an adjoint.
